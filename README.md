# Poker Odds Calculator 🧮

Аналитический инструмент на Python для расчета вероятностей выигрыша в Техасском Холдеме. Проект позволяет моделировать игровые ситуации и оценивать шансы на победу на основе текущих карт игрока и общих карт на столе.

## 🎯 Основная задача
Проект решает задачу оценки риска в условиях неопределенности. Он полезен как для игроков, желающих улучшить свою стратегию, так и для разработчиков, изучающих теорию вероятностей и комбинаторику.

## 🚀 Ключевые функции
*   **Расчет шансов (Win Rate):** Динамическое вычисление вероятности победы в зависимости от имеющихся карт.
*   **Анализ комбинаций:** Определение текущей лучшей комбинации (от старшей карты до Роял-флеша).
*   **Моделирование исходов:** Использование математических алгоритмов для оценки силы руки против случайных карт оппонентов.
*   **Визуализация:** Удобный графический интерфейс для быстрого ввода карт.

## 🛠 Технические особенности
*   **Логика правил (`Rules.py`):** Реализован сложный движок сравнения покерных рук.
*   **Статистический модуль (`StatRoom.py`):** Инструменты для сбора данных о симуляциях.
*   **Архитектура:** Чистое разделение логики расчетов и UI-слоя.

## 📂 Структура проекта
*   `Rules.py` — Сердце проекта: алгоритмы определения комбинаций и их старшинства.
*   `Gui.py` — Интерфейс для выбора карт и отображения прогресс-баров вероятностей.
*   `Players.py` — Обработка данных о картах в руках и на борде.

## 🔧 Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите приложение: `python main.py`
